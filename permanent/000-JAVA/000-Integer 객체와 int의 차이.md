#JAVA #객체지향

Integer 객체는 Wrapper클래스이다. int는 Primitive 자료형이다.

Primitive 자료형은
	1. 산술연산이 가능하다.
	2. null로 초기화 할 수 없다.

Wrapper클래스는 
	1. [[001-Boxing과 Unboxing|Unboxing]]을 하지 않으면 산술 연산이 불가능 하다.
	2. null 값을 처리할 수 있다.

Wrapper클래스는 null 값 처리가 용이 하기 때문에 SQL 및 DB에 연결하기 용이하다.
