#VO #OOP #DDD #JAVA 

VO란 도메인에서 한 개 또는 여러개의 속성을 묶어 특정한 하나의 복합물로 나타내는 객체를 의미한다.

대표적인 예시로 좌표 값이 있다. 좌표 값은 int 속성 두 개를 받아 위치를 나타내는 객체로 나타낼 수 있다.
