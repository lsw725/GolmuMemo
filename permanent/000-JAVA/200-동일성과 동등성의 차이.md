#JAVA 

동일성은 객체의 메모리 주소가 같은지를 판단한다.
\=\=연산자로 연산할 경우 동일성 검사이다.

동등성은 [[200-equals 메서드|equals 메서드]]를 통해 검사한다.
특정 속성값들을 기준으로 동등성 비교를 하기 위해서는 equals 메서드를 재정의 해야한다.
